## Description

This is my personal portfolio website for showcasing who I am as a developer, the services I offer, and a list of projects I've built. It is a fully responsive, static site built with semantic HTML and modern CSS, with smooth navigation between sections like Home, About, Services, Projects, and Contacts.

## File Structure

- `index.html` – Main HTML file containing all sections of the site (header, hero, about, services, projects, and contact form).
- `style.css` – Global stylesheet that defines the layout, typography, colors, and responsive behavior.
- `images/` – Folder for images used on the site (profile photo, project thumbnails, etc.).
- `icons/` – Folder for any icon assets used in the UI.
- `resume/` – Folder containing my downloadable resume file.
